# Facebook Post - git-vanish

🚨 **Accidentally pushed your passwords, API keys, or `.env` to GitHub?**

Don't freak out — I built the fix.

Meet **git-vanish** 👻 — a free, open-source tool that wipes a sensitive file from your _entire_ git history in seconds, while keeping the file safe on your computer.

You just run:
👉 `npx git-vanish`

A beautiful file browser opens right in your terminal. Pick the file(s) you want to erase, press Enter, confirm — and it's gone from every commit, forever. Your actual file stays untouched on disk. Nothing is lost. Git just forgets it ever existed. 🔥

---

**No technical knowledge required. No scary commands. Just keyboard arrows and Enter.**

🔧 Works on any git repo
🗂️ Multi-file selection with Space bar
🔐 Keeps your file on disk — only removes from git history
📦 Install once, use forever: `npm install -g git-vanish`

Find it on npm 👇
🔗 https://www.npmjs.com/package/git-vanish
